# What I learned
- square brackets at beginning means list []
- mixed type lists/dictionaries
- dict keys - use string keys or tuple keys
- prices = {'AAPL(2026-03-24)':229}         -string
- prices = {('AAPL', '2026-03-24'): 229}    -tuple
- set - for getting unique items from collections
- uses brackets but is not a dictionary
- search in set is much faster

- when to choose which data structure
    - do you want it t be mutable
    - do you want duplicates
    - do you want it to be ordered
    - how do you want to access it
