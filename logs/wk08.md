Strings vs lists
- both use math functions, len, print, slicing, in, index, for loop, but differently
- lists are mutable, strings are immutable
- how to sort a list in place: 
    - sorted(list) will sort list without changing order of list variable
    - list.sort will sort list variable
    - list.sort(reverse=True) to reverse order
Copying vs Aliasing
- aliasing means having multiple variables point to the same object in memory, while copying creates a new, independent object
- languages = list, if you change list, language will also change
- languages = list[:], if you change list, languages does not change
Split
- split() treats consectutive white space as one separator
- split(' ') splits each single space (empty string)
- line.strip()

Dictionary Mapping
- eng2sp = {'one':'uno, 'two':'dos', 'three':'tres'}
- a list is a sequences, a dictionary is not a sequence
- dictionaries use keys to find values, keys must be unique
- each item is a pair
- eng2sp ['four'] = 'quatro'
- len(eng2sp), prints 4
- eng2sp, prints {'one':'uno, 'two':'dos', 'three':'tres', 'four':'quatro'}
- for x in eng2sp, print x: prints one two three four (all the keys)
- for eng in eng2sp, sp = eng2sp[eng], if sp == 'dos', print(eng): prints two
- prices.keys vs prices.values
Creating and Modifying Dictionaries
- start empty: prices = {}
- add entries: prices['AAPL'] = 178.50
- update existing key: prices['AAPL'] = 182.30
- delete a key: del prices['GOOG']
- print(prices), {'AAPL', 'x', 'y'}
Counting Pattern
- keeps track of how many times each letter in a defined word/text file appears
